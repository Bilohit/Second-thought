# Second Thought Skills

Community skills installed in this project.

## Installed Skills

- `Leonxlnx/taste-skill`

## Usage

Add a new skill:
```bash
npx skills add <username>/<skill-name>
```

Remove a skill:
```bash
npx skills remove <username>/<skill-name>
```

List all skills:
```bash
npx skills list
```

## How Skills Work

Skills are loaded from the community registry and installed into `.claude/skills/`. Each skill provides additional functionality to Claude Code through custom commands and tools.
