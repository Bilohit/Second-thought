00 — THE NAME

Why "Second Thought"?

Most tools make you stop and decide — where does this go, what do I call it, how do I file it — the moment something arrives. Second Thought removes that. You leave a document, a thought, an idea, and it becomes a literal second thought: something you don't have to think about now. The deciding, organizing, and recall happen later, automatically, without your attention. Drop it and move on.

The whole idea Your first thought is the idea itself. The second thought — where it belongs, what it links to, how you'll find it again — is the one you never have to have.

03 — WHAT HAPPENS ON CAPTURE

Four stages, streamed live.

No opaque spinner. The capture overlay shows each stage as it runs — sequence communicates progress, and progress earns trust.

AI structures the note (two-pass retry), then dedup → merge or write → index.
Why it matters Categories are never hardcoded. The list of folders the AI can choose from is rebuilt from your vault on every capture. Add a folder, and the next capture can already use it.

05 — WHEN IN DOUBT, RECEDE

The scratchpad never guesses wrong.

Low confidence → scratchpad

If the AI isn't sure where something belongs, it goes to a catch-all folder instead of being confidently mis-filed. You triage it later from the Inbox.

Dedup → merge

Capture something you already have, and it merges into the existing note — based on what's actually in your files, never a hidden database.

Vision is fail-fast; everything else is fail-soft

Web / YouTube / audio failures file a placeholder so nothing is lost. But if a required image analysis fails, the capture routes straight to the scratchpad rather than letting the AI reason from nothing.

06 — FOR THE TERMINAL

Everything is scriptable.

The GUI is a convenience over a pipeline that runs just as well from the command line.

main.py 
python main.py                       # capture from clipboard
python main.py --text "..."          # inject text, skip clipboard
python main.py --url  "https://..."  # inject a URL directly
python main.py --audio path.mp3      # transcribe via local Whisper
python main.py --dry-run             # print LLM output, no vault write
python main.py --verbose             # print every pipeline stage
python main.py --self-check          # verify Ollama/vault/whisper/index
python main.py --log --stats         # summarize the capture audit log
Gotcha Keep the Ollama URL bare — http://localhost:11434, never /v1-suffixed. A /v1 in config breaks image captures with 404s.
